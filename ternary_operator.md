## В 2 предложениях

Операция-однострочная запись оператора if-else:

```
тип переменная = условие ? истина : ложь;
```

Присваивает переменной результат, а не выполняет оператор.
## Код

```c
// Синтаксис
int max  = (a > b) ? a : b;

// В макросе (безопасно)
#define ABS(x) ((x) < 0 ? -(x) : (x))

// Вложенные тернарники (сложно читать)
int max = (a > b) ? (a > c ? a : c) : (b > c ? b : c);
```
## Когда использовать

- Присваивание
- Макросы (константы)
- Флаги с одним условием
## Связи

- #написать_заметку else-if
- #написать_заметку macros_c
## Особенности для embedded

- В embedded практически не используются
