## В 2 предложениях

Работа с отдельными битами числа: установка, проверка, сброс, инверсия, сдвиг (см. [[priority]])
## Код

### Установка бита (OR)
```c
uint32_t reg |= (1 << 3);  // Бит 3 = 1, остальные без изменений
```
### Сброс бита (NOT)

```c
uint32_t reg ~= (1 << 3);  // Бит 3 = 0, остальные без изменений
```
### Проверка бита (AND)

```c
if (reg & (1 << 3)){
	// Бит 3 установлен
};
```
### Инверсия бита

```c
uint32_t reg ^= (1 << 3);  // Был 1 - стал 0, был 0 - стал 1
```
### Сдвиг бита

```c
reg << 3;  // Сдвиг на 3 бита влево
```

## Особенности для embedded
### Регистры микроконтроллера

```c
// STM32: управление GPIO
GPIOA->ODR |= (1 << 5);   // Установить PA5 в 1
GPIOA->ODR &= ~(1 << 5);  // Сбросить PA5 в 0
GPIOA->ODR ^= (1 << 5);   // Переключить PA5
```
### Атомарность

```c
// @learned Чтение-модификация-запись НЕ атомарно!
reg |= (1 << 3);  // Прерывание может вклиниться между чтением и записью

// @pattern STM32: специальные регистры BSRR (атомарно)
GPIOA->BSRR = (1 << 5);       // Установить — одна запись
GPIOA->BSRR = (1 << (5+16)); // Сбросить — одна запись
```
### Беззнаковый и знаковый сдвиги

```c
// @learned Арифметический сдвиг сохраняет знак
int8_t s = -8;     // 0b11111000
uint8_t u = 248;   // 0b11111000

s >> 1;  // 0b11111100 (-4) — знак сохраняется (арифметический)
u >> 1;  // 0b01111100 (124) — заполняется 0 (логический)

// @pattern В embedded всегда используй uintX_t
```
