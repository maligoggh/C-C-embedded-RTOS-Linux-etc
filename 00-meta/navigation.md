### Obsidian

| Что                  | Горячая клавиша   |
| -------------------- | ----------------- |
| Быстрый поиск файлов | `Ctrl+O`          |
| Граф связей          | `Ctrl+G`          |
| Новая заметка        | `Ctrl+N`          |
| Вставить шаблон      | `Alt+T`           |
| Поиск по тексту      | `Ctrl+Shift+F`    |
| Перейти по ссылке    | `Ctrl+клик`       |
| Назад/вперёд         | `Alt+←` / `Alt+→` |

**Панели:** слева — файловое дерево, справа — backlinks (кто ссылается на эту заметку).

### VS Code

| Что | Горячая клавиша |
|-----|-----------------|
| Todo Tree (твои `@learned`) | `Ctrl+Shift+G` (или иконка в боковой панели) |
| Поиск по всему проекту | `Ctrl+Shift+F` |
| Перейти к файлу | `Ctrl+P` |
| Перейти к символу | `Ctrl+Shift+O` |

### Терминал (Git)

| Что | Команда |
|-----|---------|
| Начать работу | `git pull` |
| Закончить работу | `git add . && git commit -m "..." && git push` |
| Статус | `git status` |
| История | `git log --oneline -5` |

## Типичный сценарий

**Нашёл новое про указатели:**
1. `Ctrl+N` в Obsidian → `pointers_arithmetic`
2. `Alt+T` → шаблон concept
3. Пишешь, сохраняешь (`Ctrl+S`)
4. В `stack_vs_heap` добавляешь `[[pointers_arithmetic]]`
5. `Ctrl+G` — проверяешь связь

**Пишешь код в VS Code:**
1. `// @learned 2026-03-30: арифметика указателей`
2. Сохраняешь
3. Вечером: Todo Tree → собираешь все `@learned` → в Obsidian

**Переключаешь устройство:**
1. `git push` на ПК
2. На ноутбуке: `git pull`
3. Открываешь Obsidian — всё на месте
